# Core.Packages

## 📝 Proje Açıklaması
Core.Packages, .NET projelerinde yaygın olarak kullanılan temel paketleri ve altyapı bileşenlerini içeren bir kütüphane projesidir. Bu paket, projelerde tekrar eden kod yapılarını minimize etmek ve standart bir altyapı sunmak amacıyla geliştirilmiştir.

## 🏗️ Paket İçeriği
- **Core.Application**: Uygulama katmanı için temel bileşenler
- **Core.CrossCuttingConcerns**: Çapraz kesişen işlevler (loglama, cache, vb.)
- **Core.ElasticSearch**: Elasticsearch entegrasyonu
- **Core.Persistence**: Veritabanı işlemleri için temel yapılar
- **Core.Security**: Güvenlik ve yetkilendirme bileşenleri

## 🛠️ Teknolojiler
- .NET Core
- Entity Framework Core
- Elasticsearch
- JWT (JSON Web Token)
- AutoMapper
- MediatR

## ⚙️ Kurulum
1. NuGet Package Manager üzerinden paketi projenize ekleyin:

```bash
dotnet add package Core.Packages
```

## 🚀 Özellikler
- Generic Repository Pattern implementasyonu
- Sayfalama (Pagination) altyapısı
- JWT tabanlı kimlik doğrulama
- Exception handling mekanizması
- Elasticsearch entegrasyonu
- Cross-cutting concerns (logging, caching, vb.)
- Dynamic query building
